Class Aliases

Common units aliased by their symbol.

Inheritance
Object
Aliases
Namespace:Delta.Units.Systems
Assembly:Delta.Units.dll
Syntax
public static class Aliases : object
Remarks

By inserting a 

using static Delta.Units.Systems.Aliases
at the top of a source file, the units can be used without any prefix.
